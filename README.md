# Getting start with Electron
Simple app using Electron, Node.js

## Installation and run

Make sure you have [NodeJs](https://nodejs.org/en/download/) installed

Bower:
```sh
$ npm install -g bower
```

Install bootstrapp:
```sh
$ bower install bootstrap
```

Install Dependencies
```sh
$ npm install
```

Run Server
```sh
$ npm start
```
